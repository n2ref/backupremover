# Backup remover
Delete extra backups on schedule

```
Backup remover
Usage: php backupremover.php [OPTIONS]
Required arguments:
        -d      --dir           Inspection directory
        -c      --count         Count of valid backups
Optional arguments:
        -h      --help          Help info
Examples of usage:
php backupremover.php -d /var/backups/ -c 5
```
